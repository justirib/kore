# kore
kore is a mess-around opensource baseplate made with three.js by google gemini.

# features:
- touchscreen support
- infinite jump
- camera system sorta similar to roblox except pc camera is
up = up , down = down , left = left , right = right arrow keys

# where to get kore:
kore is hosted publicly on https://kore-engine.vercel.app!


*`index.html` contains the latest source code and build of kore!
